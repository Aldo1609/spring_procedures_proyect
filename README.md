
# Spring Data JPA - Stored Procedures


This is a simple proyect with 2 endpoints:

- /verPersonas
- /buscarPersonas

## Requirements

📏 JDK 17

📏 Springboot 3.0

📏 MySQL (Or replace the driver)

* Scripts in resources!

## Endpoints examples

```bash
    ENDPOINT /verPersonas
    Type: GET

    URI: http://localhost:8080/person
```
```bash
    ENDPOINT /buscarPersonas
    Type: GET
    
    URI: http://localhost:8080/person/1
```
